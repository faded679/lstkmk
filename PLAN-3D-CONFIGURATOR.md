# 🏗️ План реализации 3D-конфигуратора зданий

**Референс:** https://evrazsteelbox.ru/configurator-3d/calc_page/
**Дата:** 1 июля 2026

---

## 📦 Что умеет референс-конфигуратор

- **3D-визуализация** здания в реальном времени (вращение, масштаб)
- **5 серий каркасов** под разные нагрузки и назначения
- **Климатические зоны:** снег, ветер, сейсмика (справочник городов РФ)
- **Крановое оборудование:** 3.2 / 5 / 10 тонн, 1 или 2 крана, подкрановые пути
- **Стены:** сэндвич-панели / профлист, толщина, горизонтальная/вертикальная раскладка
- **Кровля:** сэндвич / мембрана / профлист, выбор толщины
- **18 цветов RAL** для панелей и нащельников
- **Элементы заполнения:** окна, ленточное остекление, двери, ворота
- **Цоколь и парапет** с выбором высоты
- **СМР:** фундамент, ж/б плита пола, доставка, монтаж каркаса, монтаж ограждений
- **PDF-коммерческое предложение** за 2–3 минуты
- **Шаг 10 см** по всем габаритам (Ш×Д×В)

## 📅 План реализации (14 недель)

### Этап 1: 3D-ядро — недели 1–5
```
Неделя 1–2: Архитектура и прототип
- Выбор стека: Three.js + React Three Fiber
- Подготовка GLTF/GLB моделей для каждой серии каркаса
- Базовый рендеринг: стены, крыша, пол
- Орбитальное управление камерой (rotate, zoom, pan)

Неделя 3–4: Каркас и элементы
- Отрисовка колонн, ферм, прогонов, связей по серии каркаса
- Переключение между 5 сериями в реальном времени
- Подсветка элементов, легенда

Неделя 5: Материалы и цвета
- Текстуры сэндвич-панелей (горизонтальная/вертикальная раскладка)
- 18 цветов RAL — запекание в текстуру
- Профлист, мембранная кровля
```

### Этап 2: Расчётный движок — недели 1–4 (параллельно с 3D)
```
Неделя 1–2: База данных и формулы
- Сортамент металла (двутавр, швеллер, уголок, труба)
- Таблицы нагрузок по СП 20.13330 (снег, ветер по районам)
- Расчёт металлоёмкости по серии и габаритам

Неделя 3–4: Сложные расчёты
- Крановые нагрузки (3.2/5/10т, 1 или 2 крана)
- Подкрановые пути: балки, рельсы, упоры
- Сейсмичность (по СП 14.13330)
- Фундамент и ж/б плита пола (предварительный расчёт, точный — после геологии)
```

### Этап 3: UI/UX — недели 5–8
```
Неделя 5–6: Визард (7 шагов)
- Шаг 1: Назначение здания → выбор серии каркаса
- Шаг 2: Габариты (Ш×Д×В, шаг 10 см, ползунки + поля ввода)
- Шаг 3: Климатическая зона (город → снеговой/ветровой район)
- Шаг 4: Стены и кровля (тип, толщина, цвет)

Неделя 7–8: Завершение визарда
- Шаг 5: Краны и оборудование
- Шаг 6: Окна, ворота, двери (drag-and-drop на 3D-модель)
- Шаг 7: СМР и контакты для КП
- Адаптивная вёрстка (десктоп + планшет, мобильная версия с оптимизированным 3D)
```

### Этап 4: Интеграции — недели 9–10
```
Неделя 9: PDF и КП
- Puppeteer/Playwright → рендер страницы в PDF
- Шаблон КП с логотипом, ценами, 3D-рендером, спецификацией
- Отправка на email

Неделя 10: CRM
- Вебхук в CRM (amoCRM / Битрикс24 / своя)
- Сохранение конфигурации (восстановление по ссылке)
- История расчётов в личном кабинете
```

### Этап 5: Тестирование и запуск — недели 11–14
```
Неделя 11: Функциональное тестирование
- Все комбинации: 5 серий × габариты × климат × краны × цвета
- Граничные значения (мин/макс габариты для каждой серии)
- Валидация цен с инженерным Excel

Неделя 12: Производительность и мобильные
- Оптимизация 3D (LOD, frustum culling, сжатие текстур)
- Мобильная версия (оптимизированная 3D или статичные ракурсы)
- Lighthouse: Target > 60 FPS на среднем десктопе

Неделя 13: UAT и правки
- Тестирование с реальными менеджерами
- Сбор обратной связи, правки UI
- A/B-тест старой и новой страницы калькулятора

Неделя 14: Запуск
- Деплой на прод
- Обучение менеджеров
- Мониторинг ошибок (Sentry)
```

## 🧱 Технический стек (референс)

| Слой | Технология | Назначение |
|---|---|---|
| **Фронтенд** | React 18+ / Next.js | SPA с серверным рендерингом для SEO |
| **3D** | Three.js + React Three Fiber | 3D-визуализация каркаса |
| **Модели** | GLTF/GLB (Blender → экспорт) | 5 серий каркасов |
| **Расчёты** | Node.js / Python (FastAPI) | Серверный API расчёта нагрузок |
| **База данных** | PostgreSQL | Сортамент, цены, климатические зоны |
| **Кэш** | Redis | Кэширование расчётов |
| **PDF** | Puppeteer / Playwright | Генерация КП |
| **CRM** | Webhook → amoCRM / Bitrix24 | Передача лидов |
| **Хостинг** | VPS (4 GB RAM, 2 vCPU) | Сервер расчётов + статика на CDN |

## ⚠️ Риски

| Риск | Вероятность | Влияние | Митигация |
|---|---|---|---|
| Расчётный движок не сходится с Excel | Высокая | Критичное | Поэтапная сверка, тестовый набор из 20+ конфигураций |
| Производительность 3D на слабых ПК | Средняя | Высокое | LOD-модели, fallback на статичные рендеры |
| Сложность UI отпугивает клиентов | Средняя | Среднее | 7 шагов вместо одного экрана, кнопка «Назад» везде |
| Зависимость от BIM-моделей | Низкая | Высокое | Параметрическая генерация геометрии без готовых моделей |
