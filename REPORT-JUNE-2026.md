## 🏢 mctender.ru — Внутренняя CRM-система МАКСТИЛ

Полноценная CRM для управления продажами и клиентами, развёрнута на том же сервере.

### Функционал

| Модуль | Возможности |
|---|---|
| **Воронка заявок** | Автоматический приём лидов с lstkmk.ru (формы, чат, калькулятор) |
| **Статусы заявок** | Изменяемые статусы — полный цикл от «Новая» до «Закрыта» |
| **База клиентов** | Карточки клиентов и организаций с автозаполнением **по ИНН** |
| **Таск-трекер** | Постановка и отслеживание задач по заявкам |
| **Мониторинг чатов** | Просмотр диалогов клиентов с AI-помощником lstkmk.ru |
| **Авторизация** | Защищённый доступ (логин/пароль) |

### Интеграции

- ⬅️ **lstkmk.ru → CRM:** заявки с сайта (форма, чат AI, калькулятор) попадают в CRM мгновенно
- ⬅️ **lstkmk.ru → CRM:** все диалоги AI-чата логируются для анализа менеджерами
- ➡️ **CRM → Telegram:** уведомления о новых заявках

### Технический стек

- **Бэкенд:** FastAPI (Python)
- **Фронтенд:** SPA (React)
- **База:** SQLite / PostgreSQL
- **Хостинг:** тот же сервер (89.111.163.48), Nginx + PM2

### 2. 🖥 Разработка функционала

**Калькулятор стоимости:**
- 3 категории зданий (склад, сельхоз, сервис)
- Дискретные слайдеры ширины/длины/высоты
- 3 варианта утепления (профлист / сэндвич / без утепления)
- Ценообразование по реальной Excel-таблице (интерполяция ₽/м² от площади)
- 2×2 лэйаут: тип+утепление | чат / смета | 3D-эскиз
- Калькулятор сэндвич-панелей (отдельная страница)

**3D-конфигуратор зданий (Three.js с нуля):**
- Реалистичные ЛСТК-профили (Sigma Σ и C-профили с фланцами)
- Правильная геометрия ферм (portal frames с диагональными связями)
- Болты, анкерные болты, рёбра жёсткости на колоннах
- Интерактивная камера, стальной материал с освещением

**AI-ассистент:**
- Чат-бот на DeepSeek V4 на **всех** страницах сайта
- Голосовой ввод (Web Speech API) на мобильных
- Встроен в калькулятор — подсказывает по типу здания
- Разнообразные CTA без навязывания телефона
- Сохраняет диалоги в БД + пересылает в CRM
- Форма обратного звонка

**Каталог продукции:**
- Карточки товаров с реальными фото
- Фильтры по категориям
- Детальные страницы с характеристиками

**Лидогенерация:**
- Контактная форма → Telegram-уведомление + CRM (mctender.ru)
- SQLite база лидов + админ-панель
- Вакансии с модальными окнами и формой резюме → Telegram + CRM
